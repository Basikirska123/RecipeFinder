## Project Description
RecipeFinder is a web application.
It allows users to search for recipes, view ingredients, and instructions.
The user enters a search term, and the application displays matching dishes from a public recipe API.

## Problem Statement
Many people have a hard time figuring out what to cook. 
RecipeFinder helps you quickly find meal ideas and provides clear cooking instructions. 
This app is for those who want to quickly find new recipes for everyday life.

## API Selection
**API Name:** TheMealDB 
**API Documentation:** https://www.themealdb.com/api.php 
**Why this API?** [1-2 sentences]
TheMealDB provides easy access to recipe data, has clear documentation, 
a free apiKey option, and CORS support. This allows for quick API integration and easy work with data in a frontend environment.
